Features

The neural network can recognize multiple fruits in a single image, allowing flexible implementatons.

Twilio API Integration

When the system detects a rotten fruit, it notifies you throuhusing Twilio API.

Business Model

Making the world a better place, one apple at a time

Corporte Responsibility

Removing rotten fruits from supply chain on early stages lead to less transportation. Thus, less waste and cost.

This also allows re-use of these products through compost.

Profitability

We are charging a flat fee for each scan. In addition, we take an extra fee for each time the system detect a rotten fruit

This business plan is sustainable as well as encouraging producers to make products with high quality.

Scalability

Since the system is on cloud, it can easily be improved and implemented in any phase of suppy chain.

With iTradeNetworks image dataset, the neural network can be improved to analayze all sort of products
